## 🔑 Key Concepts Covered

- **Reference variables** — pass-by-reference to modify original values
- **Friend functions** — access private members of a class from outside
- **Constructors** — initialize objects at creation time
- **Operator overloading** — define custom behaviour for `+`, `==` on user-defined types
- **Exception handling** — `try`, `throw`, `catch` for runtime error management
- **Inheritance** — single, multiple, and virtual base classes to avoid the diamond problem
